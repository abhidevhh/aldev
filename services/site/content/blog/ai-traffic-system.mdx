# Building an AI Traffic Light Control System with YOLOv8

This project focuses on intelligent traffic flow optimization using real-time vehicle detection, adaptive signal timing, ANPR, and violation reporting.

## Core Stack
- YOLOv8
- OCR
- DeepSORT
- Streamlit
- Python
- OpenCV

## Key Engineering Challenges
- real-time vehicle tracking
- license plate recognition
- speed estimation
- signal optimization logic
- violation email alerts

## Outcome
The system reduced congestion and improved intelligent traffic automation workflows.
