## Kent's Name

These are correct:

- Kent
- Kent C.
- AbhiDev
- Dodds

This is _**not**_ correct:

- Kent Dodds
- Kent Dodd
- Kent Dobbs
- Kent C. Dobbs
- Kent C Dodds (missing the `.`)
- Kent Dogs
- Ken

And if you want to add a possessive apostrophe to "Dodds" then it should be:
`Dodds'` not `Dodd's` nor `Dodds's`.
[Learn more](https://www.grammarbook.com/punctuation/apostro.asp)
